Propriedade Opacity

Propriedade de transparencia, utilizamios bastante para formatação de imagens em nossos projetos.
Para utilizarmos a propriedade utilizamos o código da seguinte maneira, colocamos ele em casa decimais que seria de 0.1 até 1.0 que no caso seriam respectivamente:
- 0.1 = 10%;
- 0.2 = 20%;
- 0.3 = 30%;
- 0.4 = 40%;
- 0.5 = 50%;
- 0.6 = 60%;
- 0.7 = 70%;
- 0.8 = 80%;
- 0.9 = 90%;
- 1.0 = 100%;
Podemos tambem utilizar a propriedade opacity para fazermos botoes que quando passamos o mouse em cima ele traz um certo elemento visual.
Exemplo:

Botão

Como foi feito:

index.html


            <-a href="teste.html" class="exbtopacity">Botão<-/a>
obs:Sem os -.

estilo.css


            a.exbtopacity {
                padding: 10px;
                background-color: blue;
                color: aliceblue;
                text-align: center;
                text-decoration: none;
            }
            a.exbtopacity:hover {
                opacity: 0.7;
            }
A transparencia ela pega todos os elementos que estão dentro de determinada tag, como por exemplo:

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Adipisci, natus non laudantium temporibus ex obcaecati nihil quas quo doloribus, dolores porro distinctio molestias eius numquam, animi vero tenetur perferendis unde!
O nosso código ficou da seguinte maneira, criamos uma div extransparente que no caso tem um p lorem nela e fizemos o seguinte código e nosso estilo.css:

estilo.css


            .extransparente {
                padding: 30px;
                background-color: rgb(29, 80, 80);
                color:aliceblue;
                opacity: 0.5;
            }
