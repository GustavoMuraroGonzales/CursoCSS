Propriedade Position

Existem vários tipos de position oa mais usados são os: static, relative, fixed, absolut, stick.

Static

Por padrão nossas tags já vem com o static.

Realative

Utilizamos para fazer sobreposições em nossa aplicações, pois a nossa tag selecionada com o position relative ela só poderá ficar no lugar em que selecionamos e cada tag que selecionarmos com isso poderemos colocar um peso de sobreposição nela como podemos ver o exemplo abaixo:
O posicinamento do elemento relativo aonde ele está posicionado. Relativo ao ponto em que ele está podemos alterar o seu posicionaento, um exmeplo de uso seria para fazer uma sobreposição.

Um exemplo:
Primeira Div
Segunda Div
Terceira Div

Como fiz:

Index.html


            <-div class="excontainer">
                <-div class="exdiv1">Primeira Div<-/div>
                <-div class="exdiv2">Segunda Div<-/div>
                <-div class="exdiv3">Terceira Div<-/div>
            <-/div>

estilo.css


    .excontainer {
        background-color: blue;
        width: 100%;
        height: 180px;
        display: inline-block;
    }
    .exdiv1 {
        position: relative;
        left: 55px; (quntidade deslocada)
        top: 25px; (quntidade deslocada)
        width: 25%;
        height: 100px;
        background-color: yellow;
        display: inline-block;
        z-index: 1; (nivel de aparição camada) 
                        (mais baixo)
    }
    .exdiv2 {
        position: relative;
        width: 25%; 
        height: 100px; 
        background-color: green;
        display: inline-block;
        z-index: 2; (nivel de aparição camada)
    }
    .exdiv3 {
        position: relative;
        right: 55px; (quntidade deslocada)
        top: 25px; (quntidade deslocada)
        width: 25%;
        height: 100px;
        background-color: red;
        display: inline-block;
        z-index: 4; (nivel de aparição camada)
                            (mais alto)
    }
O z-index é usado para colcarmos camada em nossa alicação, sendo assim nós podemos colocar um valor de -99 ou 999. Assim tendo a possibilidade de colocar várias camadas em nosso site.

fixed

Uma propriedade que deixa o nosso item fixado em uma posição que nós queremos, independende da rolagem de nossa página ele vai ficar fixo sempre no mesmo lugar.
Um exemplo temos esse item no canto inferior direito de nossa tela, que nos leva a nossa página de teste.

Como foi feito:

Index.html


        <-a  class="exfixed" href="teste.html">
            <- svg "sua svg"><-/svg>   
        <-/a>
obs:Sem os -.

estilo.css


        .exfixed {
            position: fixed;
            bottom: 15px;
            right: 0;
            width: 300px;
        }

absolut

Utilizamos para fazer um posicionamento de div's filhas dentro do pai, com o exemplo acima, onde se quisermos colocar a nossa terceira div encostada no canto de nosso container, usariamos o absolut e colocariamos a quantidade de espaço que desejamos, porem para isso funcionar o container ou a tag pai não pode estar na position static, deve estar em qualquer outra.

Um exemplo:
Primeira Div
Segunda Div
Terceira Div

Como foi feito:

Index.html


        <-div class="excontainer2">
            <-div class="exdiv4">Primeira Div<-/div>
            <-div class="exdiv5">Segunda Div<-/div>
            <-div class="exdiv6">Terceira Div<-/div>
        <-/div>
obs:Sem os -.

estilo.css


        .excontainer2{
            position: relative;
            background-color: blue;
            width: 100%;
            height: 200px;
            display: inline-block;
        }
        .exdiv4 {
            position: relative;
            width: 25%;
            height: 20px;
            background-color: yellow;
            display: inline-block;
            
        }
        .exdiv5 {
            position: relative;
            width: 25%;
            height: 20px;
            background-color: green;
            display: inline-block;  
        }
        .exdiv6 {
            position: absolute;
            left: 200px;
            top: 50px;
            width: 25%;
            height: 100px;
            background-color: red;
            display: inline-block;
        }

sticky

Utilizamos para colar o elemento em algum lagor da nossa tela enquanto fazemos a rolagem do nosso site para baixo, podendo fazer ela ser contida somente dentro da nossa tag pai, no nosso caso o container.

Um exemplo:
Primeira Div
Segunda Div
Terceira Div

Como foi feito:

Index.html


        <-div class="excontainer3">
            <-div class="exdiv7">Primeira Div<-/div>
            <-div class="exdiv8">Segunda Div<-/div>
            <-div class="exdiv9">Terceira Div<-/div>
        <-div>
obs:Sem os -.

estilo.css


        .excontainer3{
            position: relative;
            background-color: blue;
            width: 100%;
            height: 350px;
            display: inline-block;
        }
        .exdiv7 {
            position: relative;
            width: 25%;
            height: 20px;
            background-color: yellow;
            display: inline-block;
            
        }
        .exdiv8 {
            position: sticky;
            top: 0;
            width: 25%;
            height: 190px;
            background-color: green;
            display: inline-block;  
        }
        .exdiv9 {
            position: relative;
            width: 25%;
            height: 20px;
            background-color: red;
            display: inline-block;
        }

Exemplo prático

Centralizado


Como foi feito:

Index.html


        <-div class="containerexpratico">
            <-img src="../img/fotoexemplo.jpg">
            <-div class="divexpratico">
                <-h2>Centralizado<-/h2>
            <-/div>
        <-/div>
obs:Sem os -.

estilo.css


        .containerexpratico {
            position: relative;
            display: inline-block;
            height: 400px;
            width: 100%;
            z-index: 1;
        }
        img {
            width:100%;
            opacity: 0.7;
            
        }
        .divexpratico {
            position: absolute;
            top: 40%;
            width: 100%;
            text-align: center;
            color: black;
            z-index: 999;
        }
