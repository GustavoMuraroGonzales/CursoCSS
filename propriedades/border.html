Propriedade Border

Utilizamos muito essa propriedade para deixar o nosso site mais bonito, podendo utilizar ele para fazer botão dentre outros elementos.
As tag's do border são:

estilo.css


            border: *px(tamanho)
            

            border: dashed (tracinho)
            
estilo.css

            

                .exemplo_border1 {
                    border: 1px dashed black;
                    text-align: center;
                    font-size: 15px;
                }
            

            
Exemplo dashed

            

            border: solid (linha)
            
estilo.css

            

                .exemplo_border2 {
                    border: 1px solid black;
                    text-align: center;
                    font-size: 15px;
                }
            

            
Exemplo solid

            

            border: dotted (pontinhos)
            
estilo.css

            

                .exemplo_border3 {
                    border: 1px dotted black;
                    text-align: center;
                    font-size: 15px;
                }
            

            
Exemplo dotted

            

            border: double (duas bordinhas)
            
estilo.css

            

                .exemplo_border4 {
                    border: 5px double black;
                    text-align: center;
                    font-size: 15px;
                }
            

            
Exemplo double

            

            border: groove (efeito 3d)
            
estilo.css

            

                .exemplo_border5 {
                    border: 1px groove black;
                    text-align: center;
                    font-size: 15px;
                }
            

            
Exemplo groove

            

            border: none or hidden (none nenhuma, hidden invísivel)
            
estilo.css

            

                .exemplo_border6 {
                    border: 1px none black;
                    text-align: center;
                    font-size: 15px;
                }
                or
                .exemplo_border6 {
                    border: 1px hidden black;
                    text-align: center;
                    font-size: 15px;
                }
            

            
Exemplo none or hidden

Tambem temos as opções de colocar as bordas somente de um lado, usando as mesmas condições, somente adicionado o seguinte comando no inicio de nossa tag:

 
                border-"direção": red;
Um exemplo :

estilo.css


                .exemplo_borderdirecoes {
                    border-top: 5px solid black;
                    border-left: 5px solid blue;
                    border-right: 5px solid red;
                    border-bottom: 5px solid green;
                    text-align: center;
                    font-size: 15px;
                }
Exemplo Direções
s Além das cores tambem podemos mudar os tipos como por exemplo:

estilo.css


                .exemplo_borderdirecoes2 {
                    border-top: 5px dotted blue;
                    border-left: 5px dotted blue;
                    border-right: 5px dashed red;
                    border-bottom: 5px dashed red;
                    text-align: center;
                    font-size: 15px;
                }
Exemplo Direções 2

Tambem usamos a borda arredondada em nossas aplicações.


                border-radius: "quantidade"% ou "quantidade"px;
Um exemplo:

estilo.css


                .exemplo_border_radius {
                    color:aliceblue;
                    border: 4px solid red;
                    border-radius: 30px;
                    text-align: center;
                    font-size: 15px;
                    background-color: #333;
                }
