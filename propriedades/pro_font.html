Propriedade Font

Na propriedade font podemos alterar as fontes de nosso site tambem podemos mudar o tamanho da fonte, assim deixando o nosso site mais bonito e tambem, de uma maneira melhor de se ler.

font-family

Posso alterar a fonte de meu parágrafo usando as fontes padrões que a maioria dos navegadores aceitam. Acessamos e selecionamos essas fontes da seguinte maneira:
Utilizamos a tag: font-family: e selecionamos a fonte que desejamos usar.
Vamos alterar a fonte do parágrafo abaixo:

Exemplo de alteração de fonte

Como fizemos:

Index.html


            <-p class="font1">Exemplo de alteração de fonte<-/p>

estilo.css


            .font1 {
                font-family: Verdana, Geneva, Tahoma, sans-serif;
            }

font-size

Com essa Tag podemos alterar o tamanho de nossa font de nossa aplicação. Vamos alterar o tamanho da fonte do parágrafo abaixo:

Esse parágrafo é um exemplo de mudança do tamanho fonte.

Como foi feito:

Index.html


            <-p class="fontsize">Esse parágrafo é um exemplo 
                            de mudança do tamanho fonte.<-/p>

estilo.css


            .fontsize {
                font-size: 25px;
            }

Como podemos pegar uma fonte externa para colocar em meu site?

Para isso usaremos o Google fonts, que permite utilizarmos fonts livre de direito autorais.
Para fazer isso escolhemos a fonte que desejamos usar e copiamos o código de import e colamos na primeira linha de nosso arquivos estile.css e copiamos o font-family deixado logo embaixo e colamos no lugar em que queremos utilizar como mostra o nosso exemplo abaixo:

Alteramos a fonte desse parágrafo para usar como exemplo.

Fizemos da seguinte maneira:

Index.html


        <-p class="fontchange">Alteramos a fonte desse parágrafo 
                                    para usar como exemplo.

estilo.css


         @import url('https://fonts.googleapis.com/css2?family=Young
                                            +Serif&display=swap');
            ...
        .fontchange {
             font-family: 'Young Serif', serif;
        }

Usamos essa propriedade para colocarmos um estilo em nosso site como fonte italico, obliquo. Podemos ver no seguinte exemplo:

Alteramos o estilo desse parágrafo para usar como exemplo.

font-weight

Usamos essa propriedade para aumentarmos a grossura de nossa fonte, assim podendo deixar ela em negrito, sem necessáriamente adicionarmos um código em HTML. Um exemplo:

Alteramos a grossura desse parágrafo para usar como exemplo.

Feito da Seguinte Maneira:

Index.html


            <-p class="fontchange fontweight">Alteramos a grossura 
                    desse parágrafo para usar como exemplo.<-/p>

estilo.css


            .fontweight {
                font-weight: bold;
            }
