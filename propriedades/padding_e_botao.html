Propriedade Padding e Botão

Padding é parecido com o com o margin porem, usamos ele para formatar e dar espaço em nossas text.
Usamos bastante para fazer botões.
Como exemplo vamos fazer com que o link "voltar ara a pagina inicial" se transforme em um botão.

O código do botão "VOLTAR INICIO" ficou dessa maneira:

estilo.css


            .botao_inicial {
                margin-left: 30%;
                background-color: #333;
                padding: 20px;
                border-radius: 25px;
                color: aliceblue;
                text-decoration: none;

            }
            .botao_inicial:hover {

            }
O código ".botao_incial:hover" serve para quando o mouse estiver em cima do nosso botão ele tenha um outro comando, no nosso caso, ele mudou o background para um cinza mais claro.
