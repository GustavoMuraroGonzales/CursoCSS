Propriedade Background

Podemos utilizar isso para enfeitar a nossa página HTML. background é o fundo de nossa imagem podendo colocar uma cor sólida ou podemos utilizar um propriedade chamada background-image assim podendo colocar uma imagem de fundo. Podemos assim colocar em toda a nossa página, como tambem podemos colocar ela de um tamanho que nós necessitamos, para tal usamos width e height.
Sempre lembrando da hierarquia do css, para que não tenha nenhum bug em nosso site.
As tag's de background são :

estilo.css


background-color: gray;

background-image:url(../img/fotoexemplo.jpg);

background-size:cover or contain; (cover é para adaptar a imagem 
            automaticamente, contain ele copia a imagem para 
                    completar todo o espaço da página).

background-repeat: no-repeat or repeat or repeat-y or x;(podemos 
        colocar a magem para repetir ou podemos coloar para não 
        repetir assim deixando ela em seu tamanho original.)

background-position: "escolhemos"; (podemos colocar uma 
            posição em nossa imagem assim centralizando ela 
        corretamente, colocando ela para direita, esquerda, 
            cima, baixo ou qualquer posição que quisermos).
Podemos colocar todas essas propriedade em uma só assim deixando o nosso código menor, um exemplo:

estilo.css


    background: gray url(../img/fotoexemplo.jpg) center ;
