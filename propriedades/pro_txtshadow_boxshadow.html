Propriedade TextShadow e BoxShadow

Propriedades de Sombra, utilizamos essa propriedade para colocarmos a estilização de sombra em nossos elementos.
Como por exemplo:

Exemplo com sombra

Como foi feito:

estilo.css


            .exemploshadow {
                text-shadow: 2px 2px red;
            }
Podemos tambem adicionar blur em em nossas sombras, da seguinte maneira:

Exemplo Sombra e Blur

estilo.css


            .exemploshadow1 {
                text-shadow: 2px 2px 5px blue;
            }
Na propriedade txt shadow, é bem dificil acertarmos de primeira como queremos o nosso resultado, então sempre devemos ir testando.
Podemos adicionar, duas sombras ao mesmo tempo, fazendo alguns efeitos como:

Exemplo Sombra duas cores

estilo.css


            .exemploshadow2 {
                text-shadow: 0 0 10px yellow, 2px 2px 5px green;
            }

Efeitos de sombra em caixas

estilo.css


        .exbxshadow {
            width: 100%;
            height: 150px;
            background-color: #575555;
            box-shadow: 5px 5px 5px gray;
        }
