Flexbox

Usado para deixar o nosso site responsivel, sendo mais simples, usamos para que náo seja necessário usarmos, mas para funcionar devemos sempre colocar em nosso código como display:flex.

Tipos de Flexbox que temos:

Flex-direction

Temos vários tipoe de directions que quermos, podendo coloca em colunas (column) ou linhas (Row). Como os exemplos abaixo:

Column
1
2
3
Como foi feito:

index.html


        <-div class="exflexpai">
            <-div>1<-/div>
            <-div>2<-/div>
            <-div>3<-/div>
        <-/div>
obs:Sem os -.

estilo.css


        .exflexpai {
            background-color: blue;
            display: flex;
            flex-direction:  column;
          
          }
          .exflexpai div {
            background: white;
            border: 1px solid black;
            margin: 5px;
            padding: 20px;
            text-align: center;
          }
Rows
1
2
3

        <-div class="exflexpai2">

        <-/div>

          }

Flex-wrap

Quando as nossas div`s filhas atingem o tamanho de nossa página o elemento faz com que eles continuem dentro da div pai, porem na linha de baixo, como o exemplo:

1
2
4
5
6
7
8
9
10

index.html


        <-div class="exflexpai3">
            <-div>1<-/div>
            <-div>2<-/div>
            <-div>4<-/div>
            <-div>5<-/div>
            <-div>6<-/div>
            <-div>7<-/div>
            <-div>8<-/div>
            <-div>9<-/div>
            <-div>10<-/div>
        <-/div>
obs:Sem os -.

estilo.css


        .exflexpai3 {
            background-color: blue;
            display: flex;
            flex-direction:  row;
            flex-wrap: wrap;
          
          }
          .exflexpai3 div {
            background: white;
            border: 1px solid black;
            margin: 5px;
            padding: 30px;
            text-align: center;
          }

Flex-flow

Usado para colocarmos as duas propriedades acima em uma só, em vez de colocar flex-direction e flex-wrap, usamos flex-flow: row wrap. Fazendo com o que o nosso código fique mais curto porem tenha o mesmo propósito que o código grande.

Justify-content

Faz com que os itens de nossa div, vão diretamente para um local indicado, assim facilitando o nosso código. Nós temos vários tipos de justifty-contents, onde cada um tem uma ação específica.

Justify-content: flex-left

Coloca os nossos elementos para a esquerda de nossa página, sendo esse o Justify-content padrão.

Justify-content: flex-end

Coloca os nossos elementos para a direita de nossa página.

Justify-content: center

Centraliza os nossos itens no meio da página/elementopai

Justify-content: space-between

Adiciona um espaço entre as nossa div's filhas.

Justify-content: space-around

Adiciona um espaçamento uniforme em volta de cada div filha.

Centraliza os nossos itens no meio da página/elementopai

Align-items

Usado para colocarmos posição em nossos elemento na direção vertical.

Align-items: Stretch

Estica o nosso elemento na direção vertical. Como exemplo:

1
2
3
Align-items: center

Centraliza o nosso elemento na vertical

Align-items: flex-start

Coloca o nosso elemento na parte de cima.

Align-items: flex-end

Coloca o nosso elemento na parte de baixo.

Align-content

Utilizado para alinhar todos os nosso itens filhos de uma maneira mais ordenada e bonita de ver.

Flex-grow

Faz as divisões de nossas div's de uma forma uniforme.
