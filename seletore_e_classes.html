Seletores e Classes

Temos alguns tipos de Seletores e Classes como :

  • body
    • body é um seletor onde podemos controlar todas as propriedades css de nosso site, assim colocando a cor de fundo geral que queremos dentre outras opções.
    Usamos ele da seguinte maneira:

    estilo.css

     
     body {
            background-color: red;
     }
    Assim mudando a cor do fundo de toda a página.
  • class
    • class é uma classe, toda tag que colocamos uma class nela, colocamos um localizador onde podemos adicionar um código css nessa classe, assim adicionado ela em uma tag automaticamente já seria formatado da maneira que eu fiz a class na página css.
    São comportamentos padrões que podemos utilizar ao longo de nosso código.
    Podemos pré estabelecer padrões
    Obs:Nunca deve se começar com um numero.
    É acumulativa, podemos colocar varias class achamos necessário em nossas tag's
    Podemos usar como demonstra o exemplo abaixo:

    Index.html

    
        <-div class="title">
            <-h1>Seletores e Classes<-/h1>
        <-/div>

    estilo.css

    
        .title {
            color:red;
            text-align: center;
        }
    Assim podemos alterar somente a tag seleciona, semelhente o nosso titulo.
  • id
    • id é um seletor onde damos um nome para determindada tag, assim alteramos somente ela, e podemos localiza=la.
    Cada identificador é sempre único.
    Obs:Nunca deve se começar com um numero.
    Podemos analisar o exemplo a seguir:

    Index.html

    
        <-p id="exemplo" class="center">Index.html<-/p>

    estilo.css

    
        #exemplo {
            background-color: #ccc;
        }
