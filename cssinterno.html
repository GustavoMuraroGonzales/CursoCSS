CSS Interno

É quando damos instruções CSS no ínicio de nosso código, quando criamos uma tag Style dentro da tag head, então ali colocamos os nosso comandos de propriedades de CSS.
Sempre lembrando do nível de prioridade que é:

  1. Inline;
  2. Interno;
  3. Externo.

Como demonstro no código abaixo, um exemplo de nossa página inicial:

        
       <-head>
       <-style>
        *{
             box-sizing: border-box;
         }
         body{
             margin: 0;
         }
        .menu{
             background-color: #333;
             overflow: hidden;
             border-radius: 30px;
         }
        .menu a{
             color:white;
             padding:7px 10px;
             float: left;
             display:block;
             text-decoration: none;
         }
        .topo {
             background-color: antiquewhite;
             padding: 30px;
             text-align: center;
         }
        .topo h1{
             font-size: 50px;
         }
        .paragrafo1 {
             background-color: antiquewhite;
             text-align: center;
             padding: 5px;
         }
       <-/style>
       <-/head>
