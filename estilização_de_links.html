Estilização de Links

Utilizamos a estilização de links para fazermos botões mais elaborados para o nosso site, assim deixando ele mais elegante e prazeroso em utilizar.
Para a criação desses botões no arquivo estilo.css utilizamos o seguinte comando a.btexemplo1 para que chamar a nossa classe somente em um link, se colocarmos ela em qualquer outro lugar ela não irá funcionar pois está cadastrada como link.
Exemplo de estilização de links:


Exemplo 1

como foi feito:

Index.html


            <-a class="btexemplo1" href="/propriedades/teste.html">
                                                    Exemplo 1<-/a>

estilo.css


            a.btexemplo1 {
                background-color: darkgreen;
                color: aliceblue;
                padding: 10px;
                text-decoration: none;
                border: 3px solid greenyellow;
                border-radius: 5px;
            }
            a.btexemplo1:hover {
                background-color: rgb(22, 230, 22);
                padding: 12px;
                border: 3px solid rgb(85, 136, 9);
            }

Exemplo 2

como foi feito:

Index.html


            <-a class="btexemplo2" href="/propriedades/teste.html">
                                                    Exemplo 2<-/a>

            a.btexemplo2 {
                text-decoration: none;
                padding: 10px;
                border: 1px solid blue;
                color:blue
            }
            a.btexemplo2:hover {
                text-decoration: underline;
                padding: 8px;
                color:aliceblue;
                background-color: blue;
            }
