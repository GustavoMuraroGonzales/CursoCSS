Regra Important

Ajuda a determinar qual a hierarquia para o nosso navegador reconhecer.
Utilizamos essa regra da seguinte maneira, colocamos !important na frente da tag que nos queremos que seja utilizada como exemplo:

estilo.css


            #meuid {
                color:green;
            }

            .minhaclasse {
                color:blue;
            }
            
            h1 {
                color:red; !important
            }
Na teoria, deveria o nosso titulo ficar na cor verde, porem como colocamos o nossa color no h1 como important ele irá utiliza-lá como a cor selecionada.
Mas não devemos utilizar essa regra sempre, somente quando for realmente necessário.
Mesmo quando colocamos cor em nossa tag no proprio arquivo html, ela não irá funcionar por conta do important em nosso arquivo css externo.
