Efeito Gradiante

O efeito gradiente é quando fazemos a transição de uma cor para a outra dentro de um elemento como exemplo:


Como foi feito:

Index.html


            <-div id="exgradiente">

            <-/div>
obs:Sem os -.

estilo.css


            #exgradiente {
                height: 250px;
                width: 100%;
                background-image: linear-gradient(blue, red );
              }
Se quisermos mudar a posição de onde vem para onde vai as nossas cores usamos o seguinte código.

Como foi feito:

estilo.css


        #exgradiente2 {
            height: 250px;
            width: 100%;
            background-image: linear-gradient(to left, blue, red );
        }
Tambem temos outro tipo de gradiente que é o radial, que fica em forma de circulo a nossa gradiente, igual o exemplo abaixo;

Como foi feito:

estilo.css


            #exgradiente3 {
                height: 250px;
                width: 100%;
                background-image: radial-gradient( blue, red );
              }
Tambem podemos por a porcentagem da quantidade que queremos de cada cor, como por exemplo:

Como foi feito:

estilo.css


            #exgradiente4 {
                height: 250px;
                width: 100%;
                background-image: radial-gradient( blue 50%, red  );
              }
