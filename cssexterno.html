CSS Externo

Considerado um dos modos mais produtivos e de melhor leitura do código CSS.
Nós podemos adicionar várias páginas usando somente uma página de estilo.css, utilizando o a tag de rastreio no head de cada página, igual demonstro no exemplo abaixo:

        
        <-head>
            <-link rel="stylesheet" href="estilo.css">
                                     |-> (endereço de sua página CSS).
        <-/head>
obs:Sem os -.

E o nosso código na página estilo.css ficou assim:

    
        * {
            background-color: antiquewhite;
        }
        .title {
            color:red;
            text-align: center;
        }
        .mark { /*utilizado somente para destacar a frase obs:sem os -.*/
            background-color:#00FF00;
        }
