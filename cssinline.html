CSS Inline

CSS Inline é quando nós colocamos o código de CSS juntamente com o nosso código de HTML, na mesma tag para que ela de atribuições ao nosso site, sem necessáriamente criarmos um arquivo externo.
Um exemplo abaixo seria:
Quero mudar a cor de fundo do meu site, no caso eu deveria fazer o seguinte se eu quiser usar o modo CSS Inline:
 <-!DOCTYPE html>
 <-html lang="pt-br">
  <-head>
    <-meta charset="UTF-8">
    <-meta name="viewport" content="width=device-width, initial-scale=1.0">
    <-title>CSS Inline<-/title>
  <-/head>
  <-body <-mark>style="background-color:antiquewhite">
    <-h1 style="color: red" >CSS Inline
  <-/body>
 <-/html>

No caso do CSS Inline é mais para os projetos simples onde não depende ou não necessita de grandes objetos visuais, onde você precise somente trocar a cor, fonte, alinhar. Ele é recomendado para realização de modificaçãoes pequenas.
